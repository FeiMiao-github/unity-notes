C#类型分类
	1. 值类型：存储数据本身
	2.引用类型：存储数据的引用

1.数据类型：
	1.值类型
		1.结构
			1.数值类型
			2.bool
			3.char
		2.枚举
	2.引用类型
		1.接口
		2.类
			1.string
			2.Array
			3.委托

2.值类型和引用类型：
	值类型:
		声明在栈中，数据在栈中
	引用类型：
		声明在栈中，数据在堆中

3.值类型->Object 	装箱操作（box）
	int a = 1；
	object o = a;
	在堆中开辟空间，(1, 同步块索引，类型对象指针)

4. Object->值类型	拆箱操作 （unbox）
